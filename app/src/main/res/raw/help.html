vmate filecopy (2020-09-25)

Function : Convert vmate files into valid date time format

o Current Snoppa vmate app creates media file including colon(:) and timezone is not local.
o Some video play app in android phone can NOT understand this colon, and spit out that video
o This app will copy vmate media files into Camera Folder (DCIM/vmate) with proper name.

How to operate

o When this app is loaded, it will display all the file list you downloaded from vmate gimbal.

o On Setting screen, you may set time zone to fit your local time, and may set switch for delete vmate file or not
If delete switch is on, whenever each files in source folder is copied into destination folder, it will be deleted
It can be reload this file from vmate sdcard via snoopa vmate app. o After setting, you may check both files displayed in heading area

o If ready, click play icon, then copying will be started while displaying current progress.
     from : \Vmate\sd/DCIM/100HSCAM
     to : DCIM\vmate

Notice

o No guarantee for improving functions
o This app copies vmate file into camera folder not renaming itself
o This app will disparate when snoppa vmate gives proper file name.
